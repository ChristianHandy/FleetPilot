ℹ Remote Host Requirements

The remote PC must have liquidctl installed and accessible:

# Debian/Ubuntu
sudo apt install liquidctl

# Or via pip
pip3 install liquidctl

# Allow non-root access (recommended)
# Create udev rule: /etc/udev/rules.d/71-liquidctl.rules
# SUBSYSTEMS=="usb", ATTRS{idVendor}=="1b1c", TAG+="uaccess"
# sudo udevadm control --reload-rules && sudo udevadm trigger

# Or run as root (simpler, set username=root in FleetPilot)

FleetPilot connects via SSH and runs liquidctl --match '<match_str>' initialize then status --json every 30 seconds.
