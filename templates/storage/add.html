ℹ API Key Setup

TrueNAS SCALE / CORE
1. Log in to the TrueNAS web UI
2. Go to System → API Keys
3. Click Add, give it a name, copy the key
4. The key grants full API access — store it securely
Unraid 6.12+
1. Log in to the Unraid web UI
2. Go to Settings → Management Access → API Keys
3. Create a new key with appropriate permissions

For older Unraid versions, the JSON API plugin is required.
