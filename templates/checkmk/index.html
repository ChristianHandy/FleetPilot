📋 Setup Instructions

  

  


    
    

      
      
      
      
    


    
    

      

        Place this script on the {{ _('CheckMK monitoring host') }} (or any host monitored by CheckMK).
        It queries FleetPilot and outputs CheckMK-compatible service lines.
      

      
    
        
  1. Download the script: ⬇ fleetpilot_check
    2. 
        
  2. Copy to CheckMK agent local directory:
    
          cp fleetpilot_check /usr/lib/check_mk_agent/local/
    3. 
        
  3. Make executable:
    
          chmod +x /usr/lib/check_mk_agent/local/fleetpilot_check
    4. 
        
  4. {{ _('Run service discovery in CheckMK — FleetPilot services will appear automatically') }}
    5. 
      

      

        View script source
        
{{ script_local }}

      

    


    
    

      

        Use this script to create {{ _('piggyback data') }} — each FleetPilot host appears as a separate
        host in CheckMK with its own services (updates, disks, etc.).
      

      
    
        
  1. Download: ⬇ fleetpilot_piggyback
    2. 
        
  2. cp fleetpilot_piggyback /usr/lib/check_mk_agent/local/
    3. 
        
  3. chmod +x /usr/lib/check_mk_agent/local/fleetpilot_piggyback
    4. 
        
  4. In CheckMK: enable {{ _('Piggyback') }} for the monitoring host
    5. 
        
  5. {{ _('Run service discovery — each FleetPilot host gets its own CheckMK host') }}
    6. 
      

      

        View script source
        
{{ script_piggyback }}

      

    


    
    

      

        Configure FleetPilot as a {{ _('datasource program') }} in CheckMK.
        CheckMK will call the URL directly instead of using an agent.
      

      
    
        
  1. In CheckMK: {{ _('Setup → Hosts → [your host] → Monitoring agents → Datasource programs') }}
    2. 
        
  2. Set command to:
    
          curl -sf -H "X-FleetPilot-Token: {{ token }}" {{ base_url }}/api/checkmk/agent
    3. 
        
  3. {{ _('Activate changes and run service discovery') }}
    4. 
      

      

        {{ _('Direct URL:') }}

        {{ base_url }}/api/checkmk/agent?token={{ token }}

        other monitoring tools.
